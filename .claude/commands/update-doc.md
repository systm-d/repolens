# Update Documentation Command

Met à jour la documentation et synchronise le Wiki.

## Steps

1. **Analyser les changements**
   - Comparer le code avec la documentation existante
   - Identifier les nouvelles fonctionnalités non documentées
   - Détecter les incohérences

2. **Mettre à jour les fichiers locaux**
   - `README.md`: Usage, installation, exemples
   - `DEVELOPMENT.md`: Guide développeur
   - `CHANGELOG.md`: Historique des changements
   - `wiki/`: Pages Wiki locales

3. **Valider**
   - Vérifier les liens
   - Tester les exemples de code

## Fichiers à Vérifier

| Fichier | Contenu |
|---------|---------|
| README.md | Installation, usage, exemples |
| DEVELOPMENT.md | Setup dev, architecture, contribution |
| CHANGELOG.md | Versions, changes |
| wiki/Home.md | Page d'accueil Wiki |
| wiki/Installation.md | Guide installation |
| wiki/Configuration.md | Options de config |

## Output

- Liste des fichiers mis à jour
- Différences appliquées
